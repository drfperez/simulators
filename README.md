# Simulators

An educational collection of **four interactive simulators** designed to make core topics in mechanics, electronics, and digital logic more visual, intuitive, and meaningful for students and teachers.

This repository brings together:

- **GearLab** — learn how gears transmit motion
- **ResistorLab** — decode and encode resistor color bands
- **LogicLab** — move from truth tables to Karnaugh maps, Boolean expressions, and circuits
- **GateLab** — explore logic gates through direct interaction

These tools were created to support **active learning**, **step-by-step understanding**, **visual reasoning**, and **formative assessment**.

---

## Overview

Many students can memorize formulas, procedures, or symbol systems without fully understanding the concepts behind them. These simulators were designed to bridge that gap by combining:

- dynamic visualization
- direct manipulation of variables
- step-by-step explanations
- multilingual support
- teacher-oriented guided activities

Together, they help learners connect **theory and practice** across several technical domains.

## Included Simulators

### 1. GearLab
**Interactive simulator for gear systems**

GearLab helps students understand how mechanical transmission works by letting them modify key variables such as:

- number of teeth
- module
- input speed

Students can immediately observe the effects on:

- rotation direction
- output speed
- transmission ratio
- ideal torque
- center distance

**Educational value:**
- Supports prediction and hypothesis testing
- Connects formulas with visual motion
- Helps explain why larger/smaller gears change speed and torque
- Useful for contexts such as bicycles, gear trains, clocks, and industrial mechanisms

### 2. ResistorLab
**Interactive simulator for resistor color codes**

ResistorLab supports both directions of learning:

- **color code → resistance value**
- **resistance value → color code**

It includes support for:

- 4-band resistors
- 5-band resistors
- tolerance interpretation
- minimum and maximum value ranges

**Educational value:**
- Moves beyond memorizing color tables
- Explains significant digits, multiplier, and tolerance
- Helps students understand that real resistor values fall within a valid interval
- Makes the structure of resistor coding visible and easier to reason about


## Try it out

### 3. LogicLab
**From truth tables to circuits**

LogicLab is a digital logic environment that connects multiple representations of the same Boolean function:

- truth tables
- Karnaugh maps
- canonical SOP/POS expressions
- simplified Boolean expressions
- gate-level circuits

The simulator allows students to:

- define functions with up to four variables
- edit outputs directly in the truth table or Karnaugh map
- use don't-care conditions
- simplify expressions visually
- generate and simulate logic circuits

**Educational value:**
- Makes equivalence between representations visible
- Helps learners understand simplification, not just perform it
- Connects algebraic form to circuit implementation
- Supports error checking, comparison, and reasoning


## Try it out

[Test the Logiclab simulator](https://drfperez.github.io/logiclab)

### 4. GateLab
**Fast and visual introduction to logic gates**

GateLab is a lightweight simulator for exploring how logic gates behave through click/touch interaction.

It includes:

- two interactive inputs
- a selectable gate
- immediate visual output

Available gates:

- AND
- OR
- XOR
- NAND
- NOR
- XNOR
- NOT

**Educational value:**
- Ideal for first contact with digital logic
- Encourages experimentation and pattern discovery
- Helps students intuitively understand gate behavior before formal truth tables
- Works well on computers, tablets, and projected classroom displays

## Teacher Mode

GearLab, ResistorLab, and LogicLab include or support a **teacher mode** designed for guided learning and formative assessment.

Typical workflow:

1. **Predict** what will happen
2. **Test** the idea in the simulator
3. **Explain** the result with guided feedback

Teacher-oriented features may include:

- guided questions
- answer checking
- "show solution" support
- immediate feedback
- written predictions and reasoning

This approach encourages metacognition and classroom discussion.

## Pedagogical Approach

These simulators were conceived not as isolated calculators, but as **interactive learning environments**. Their design is based on four main principles:

1. **Meaningful interaction** — learners manipulate variables and observe consequences
2. **Visual scaffolding** — abstract relationships become visible
3. **Step-by-step explanation** — observation is connected to conceptual understanding
4. **Formative assessment** — prediction, feedback, and revision are part of the learning process

The goal is not only to obtain correct answers, but to help learners:

- explain what happens
- predict outcomes when variables change
- compare representations
- transfer understanding to new situations
